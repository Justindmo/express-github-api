# Express Github API

- NodeJS
- Express
- Axios
- [https://developer.github.com/v3/](Github API)

### Usage

Input username as a parameter via URL.

```
http://localhost:8000/:username
```

### Return data

```
{"response":{"username":"justindmo","followers":10}}
```

### Start API server

```
yarn
yarn start
```

Access API via: http://localhost:8000
